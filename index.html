IMAGE DESTROYER

Image destroyer coverts images to a grid of div elements. It reads pixel color values from a canvas, converts them to grascale and reduces the bit depth of each color channel from 256 posible values to 128. Each time an image is drawn to the "div display" it introduces some random artefacts. Text is displayed without the use of a font by coloring individual "pixels" in this div grid. While this is all computationally expensive, it sure looks cool!

The text and image content is pulled from a NASA image API
